Processing Settings
Scan videos modified within the last N hours (default: 24). Use a larger value on first run to catch existing footage. For example: 1 Week = 168 hrs, 1 Month = 730 hrs, 1 Year = 8760 hrs.
Extract 1 frame per N frames. Lower = more thorough but slower. Default: 30 (~1 fps at 30 fps video).
MegaDetector confidence cutoff. Lower catches more but increases false positives. Default: 0.2.
ISO country code for SpeciesNet geographic filtering (e.g. US, GB, AU, CA).
Narrows species candidates to those with occurrence records in your state or province — eliminates species that don't live in your area (e.g. European deer in North America). Format: country-region (e.g. US-UT, US-CA, US-TX, GB-ENG). Leave blank for country-level filtering only.
The date format your cameras embed in filenames. Used to correctly filter videos by date and set recorded_at in the database. Auto-detect tries all known patterns.

Retention Policy

Set limits for how long video files are kept on the NAS. DB records (detections, species, timestamps) are always kept — only the video files are deleted. The first limit hit triggers a purge. Set to 0 to disable a limit. Blank videos are archived to a blanks folder on the NAS and deleted when retention limits are reached.

Blank Videos (no animal/person detected)
Delete blank video files older than N days. Blank videos are archived to the NAS blanks folder and kept until this limit is reached. Default: 90.
Purge oldest blank DB records when the recorded total size of blank footage exceeds this limit. Default: 20 GB.
Kept Videos (animal/person detected)
e.g. 730 = keep 2 years of footage. Default: 730. Set 0 to keep forever.
Delete oldest kept videos when total exceeds this size. Default: 500 GB.
